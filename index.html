Что такое строка?
Неизменяемая последовательность символов Unicode
Создание строк

Строка — это тип данных str. Создаётся в одинарных, двойных или тройных кавычках. Строки неизменяемы: изменить символ «на месте» нельзя.

create.py
# Три вида кавычек
s1 = 'Привет'           # одинарные
s2 = "Hello, World!"   # двойные
s3 = """Многострочный
текст"""             # тройные

print(type(s1))     # <class 'str'>
print(len(s1))      # 6
⚠️

Immutable! s[0] = 'A' вызовет TypeError. Любая операция создаёт новую строку.

Итерация по строке

Строка — это последовательность, по которой можно пройтись циклом for. Каждая итерация даёт один символ.

iterate.py
for char in "Hello":
    print(char)
# H
# e
# l
# l
# o

# Перебор с индексами — enumerate()
for i, ch in enumerate("Hi"):
    print(f"[{i}] = '{ch}'")

️ Сравнение строк

Строки сравниваются лексикографически — символ за символом по их Unicode-кодам.

compare.py
str1 = "apple";  str2 = "banana";  str3 = "apple"

print(str1 == str2)   # False
print(str1 == str3)   # True
print(str1 <  str2)   # True  ('a' < 'b' по Unicode)
print("Z" > "a")      # False (Z=90, a=97)

Форматирование строк

            
Три способа вставить данные в строку

          

        


        

          
Три подхода

          

            
            
            
          

          

            
fstring.py

            
name = "Анна"; age = 19; gpa = 4.85

# f-строка — Python 3.6+, рекомендуемый способ
msg = f"Студент {name}, возраст: {age}, балл: {gpa:.1f}"

# Выражения прямо внутри!
print(f"Через 5 лет: {age + 5} лет")

          

          

            
format.py

            
name, age = "Анна", 19

# По позиции
"Студент {}, возраст: {}".format(name, age)

# По имени
"Студент {name}, возраст: {age}".format(name=name, age=age)

          

          

            
percent.py

            
name, age = "Анна", 19

# Старый стиль (Python 2), не рекомендуется
"Студент %s, возраст: %d" % (name, age)
# %s — строка, %d — целое, %f — вещественное

Unicode, ord() и chr()

            
Каждый символ — это число

          

        


        

          
Таблица ASCII-символов

          
Каждый символ имеет числовой код. Наведи — увидишь dec и hex.

          

        


        

          
↔️ ord() и chr()

          
unicode.py

          
ord('A')    # → 65
ord('а')    # → 1072  (кириллица)
ord('❤')    # → 10084

chr(65)     # → 'A'
chr(1040)   # → 'А'
chr(128512) # → '😀'

          

            

              
ord( символ ) → код

              

              

            

            

              
chr( число ) → символ

              

              

            

          

        


        

          
Шифр Цезаря

          
Сдвигаем код каждого символа на N позиций — наглядный пример ord() + chr().

          
caesar.py

          
def caesar(text, shift):
    return "".join(chr(ord(c) + shift) for c in text)

print(caesar("Hello", 1))  # → "Ifmmp"
